# Changelog

## [1.0.0] - Initial Release

### Реализовано

- ✅ Многораундовая система аукционов
- ✅ Атомарные операции со ставками (MongoDB транзакции + Redis locks)
- ✅ Anti-sniping механизм (продление аукциона при поздних ставках)
- ✅ Полный ledger финансовых операций
- ✅ WebSocket для live-обновлений ставок
- ✅ Фоновые задачи (BullMQ) для завершения раундов
- ✅ Rate limiting для защиты от спама
- ✅ Минимальный UI на React
- ✅ Нагрузочное тестирование (k6)
- ✅ Боты для автоматического тестирования
- ✅ Docker контейнеризация
- ✅ Подробная документация

### Известные ограничения

- Rate limiter in-memory (не распределённый)
- Упрощённая аутентификация (для демо)
- Комиссия платформы начисляется продавцу (технический долг)
- WebSocket не масштабируется горизонтально (нужен Redis adapter)
