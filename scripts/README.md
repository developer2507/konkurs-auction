# Скрипты для тестирования

## Создание тестовых данных

```bash
npm run create-test-data
```

Создаёт:
- 10 тестовых пользователей с балансом 100000
- 2 тестовых аукциона (1 активный, 1 запланированный)

## Запуск ботов

```bash
npm run bot <auctionId> [botCount]
```

Параметры:
- `auctionId` - ID аукциона (обязательно)
- `botCount` - количество ботов (по умолчанию 5)

Пример:
```bash
npm run bot 507f1f77bcf86cd799439011 10
```

Боты автоматически:
- Пополняют баланс до 100000
- Делают ставки с интервалом 2-5 секунд
- Вероятность ставки: 30% на каждом интервале
- Диапазон ставок: от минимальной до 5000
